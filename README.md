# nuxt-todo-app
#### this is my first nuxt app.

## Setup

Make sure to install the dependencies using [pnpm](https://pnpm.io/):

```bash
pnpm i
```

## Development

Start the development server on http://localhost:3000

```bash
npm run dev
```
